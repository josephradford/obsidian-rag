# Technical Notes

## DFMEA Process
Design Failure Mode and Effects Analysis (DFMEA) is a systematic method to:
- Identify potential failure modes in design
- Assess risk through severity, occurrence, and detection ratings
- Prioritize corrective actions to reduce risk

## Medical Device Software
IEC 62304 defines software lifecycle processes for medical device software:
- Software planning and development
- Risk management integration
- Configuration management
- Problem resolution
