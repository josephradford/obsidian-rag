## Sprint Planning
Sprint planning is a Scrum ceremony where the team commits to work for the upcoming sprint. It involves:
- Reviewing the product backlog
- Selecting items for the sprint backlog
- Breaking down work into tasks

## Docker Networking
Docker provides several networking options:
- Bridge networks for container communication
- Host networking for performance
- Overlay networks for multi-host setups

## Agile Practices
Key Agile practices include:
- Daily standups for team sync
- Retrospectives for continuous improvement
- Sprint reviews for stakeholder feedback
